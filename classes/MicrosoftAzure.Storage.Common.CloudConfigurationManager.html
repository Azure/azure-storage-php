registerSource()

registerSource(string  $name, callable  $provider = null, boolean  $prepend = false) : void

Registers a new connection string source provider. If the source to get registered is a default source, only the name of the source is required.

Parameters

string $name

The source name.
callable $provider

The source callback.
boolean $prepend

When true, the $provider is processed first when calling getConnectionString. When false (the default) the $provider is processed after the existing callbacks.
