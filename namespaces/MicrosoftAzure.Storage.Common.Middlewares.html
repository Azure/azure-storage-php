\MicrosoftAzure\Storage\CommonMiddlewares

Interfaces

IMiddleware IMiddleware is called before sending the request and after receiving the response.

Classes

HistoryMiddleware This class provides the functionality to log the requests/options/responses.
MiddlewareBase This class provides the base structure of middleware that can be used for doing customized behavior including modifying the request, response or other behaviors like logging, retrying and debugging.
MiddlewareStack This class provides the stack that handles the logic of applying each middlewares to the request or the response.
RetryMiddleware This class provides the functionality of a middleware that handles all the retry logic for the request.
RetryMiddlewareFactory This class provides static functions that creates retry handlers for Guzzle HTTP clients to handle retry policy.
